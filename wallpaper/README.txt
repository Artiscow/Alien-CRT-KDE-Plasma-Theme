WEYLAND-YUTANI CRT — desktop wallpapers
=======================================

Handmade original artwork by the theme's author. The whole boot theme
(GRUB menu, boot splashes, Plasma splashes) is built on this CRT texture.
Every image here is 5120x1440 (32:9) and crops/scales to any screen —
KDE handles the fit.

STILL IMAGES
  background.png              the master — green CRT phosphor + logo (default)
  weyland-crt-nologo.png      the same texture, no logo
  weyland-crt-textured.png    softer texture with a subtle moire/vignette
  weyland-crt-bright.png      brighter green gradient, no logo
  weyland-crt-alt.png         alternate logo version, brighter corners

  Set one from the terminal (KDE Plasma 6):
      plasma-apply-wallpaperimage ~/…/wallpaper/background.png
  ...or right-click the desktop -> Configure Desktop and Wallpaper... ->
  Wallpaper type: Image -> + Add Image... -> pick a file above.

ANIMATED (looping video)
  weyland-crt-loop.mp4        seamless 10s loop — a soft green phosphor sweep
                              drifts down the CRT texture (H.264, 5120x1440)

  KDE has no built-in video wallpaper, so play it with a video-wallpaper
  method of your choice, e.g. the "Smart Video Wallpaper Reborn" plugin
  (KDE Store), then point it at weyland-crt-loop.mp4. It loops perfectly,
  so no seam is visible.
