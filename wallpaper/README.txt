WEYLAND-YUTANI CRT — desktop wallpaper
======================================

Handmade original artwork by the theme's author. The whole boot theme
(GRUB menu, boot splashes, Plasma splashes) is built on this piece.

Master file:  weyland-yutani-5120x1440.png   (32:9 ultrawide; scales/crops
              to any screen — KDE handles the fit)

Set it in KDE Plasma:
  Right-click the desktop -> Configure Desktop and Wallpaper... ->
  Wallpaper type: Image -> + Add Image... -> pick this file.
