WEYLAND-YUTANI CRT — Nobara boot theme
======================================

Open  index.html  in a browser to preview the boot splashes and pick one; it
shows the exact commands for your choice. The full steps are below.
--------------------------------------------------------------------

Contents:
  grub/                     the GRUB boot-menu theme
  plymouth/weyland-muthur   boot splash: MU-TH-UR ship computer startup self-test
  plymouth/weyland-online   boot splash: the Weyland-Yutani emblem powers on + ONLINE
  plymouth/weyland-nostromo boot splash: emblem + power gauge energizing 0->100%
  plymouth/weyland-ignition boot splash: minimal — the Weyland-Yutani emblem powers on
  plymouth/weyland-console  boot splash: typed green log + blinking cursor
  plymouth/weyland-bar      boot splash: progress bar
  plymouth/weyland-standby  boot splash: segmented "standby" meter
  (every splash has the rolling CRT scanlines + sweep band — optional, see VARIANTS)
  optimized/                a 14x-smaller copy of every 5120x1440 background (see VARIANTS)

Run everything from inside this unzipped folder (cd into it first).

--------------------------------------------------------------------
VARIANTS — two optional switches (index.html sets these for you)
--------------------------------------------------------------------
The command blocks below install the DEFAULT build: native-resolution
backgrounds WITH the CRT line. Two independent switches change that; the
index.html picker adds the extra lines automatically, or apply by hand:

* Backgrounds — NATIVE vs OPTIMIZED
  Every background is 5120x1440 and crop-fits to any screen. "Native" are
  full-quality PNGs (~4 MB each); "Optimized" look identical but are ~14x
  smaller (~290 KB), which also shrinks the boot image. To use the optimized
  set, copy it over the theme's background AFTER the cp step and BEFORE the
  activation step (plymouth-set-default-theme / grub2-mkconfig), e.g.:
    boot splash: sudo cp optimized/plymouth/<name>/background.png /usr/share/plymouth/themes/<name>/background.png
    GRUB:        sudo cp optimized/grub/background.png /boot/grub2/themes/weyland-crt/background.png
    Plasma:      cp optimized/plasma-splash/<name>/contents/splash/images/background.png ~/.local/share/plasma/look-and-feel/<name>/contents/splash/images/background.png

* CRT line — ON vs OFF
  To drop the moving CRT overlay (scanline grille + sweep band) from a splash,
  flip its flag AFTER copying it in:
    boot splash: sudo sed -i 's/^crt = 1;/crt = 0;/' /usr/share/plymouth/themes/<name>/<name>.script
    Plasma:      sed -i 's/property bool crt: true/property bool crt: false/' ~/.local/share/plasma/look-and-feel/<name>/contents/splash/Splash.qml
  (GRUB has no CRT overlay, so this doesn't apply there; the Plasma "System
  Online" splash is already clean.)

* Splash size — SMALL / MEDIUM / LARGE
  Scales the CONTENT (console text, progress bar, power gauge, standby meter, the
  ONLINE text) — the Weyland emblem stays a fixed size, EXCEPT Emblem Ignition,
  whose whole content is the emblem. Default is MEDIUM. Set it AFTER copying in
  (0.8 = Small, 1.25 = Large):
    boot splash: sudo sed -i 's/^scale = 1.0;/scale = 0.8;/' /usr/share/plymouth/themes/<name>/<name>.script
    Plasma:      sed -i 's/property real uiScale: 1.0/property real uiScale: 0.8/' ~/.local/share/plasma/look-and-feel/<name>/contents/splash/Splash.qml
  Notes: the Nostromo and System-Online BOOT splashes bake the emblem into their
  animation frames, so they have no scale line (their emblem must stay fixed); their
  Plasma versions still scale the gauge / ONLINE text. GRUB uses its own menu-font
  variants above, not this scale.

--------------------------------------------------------------------
1) GRUB MENU
--------------------------------------------------------------------
sudo rm -rf /boot/grub2/themes/weyland-crt
sudo mkdir -p /boot/grub2/themes
sudo cp -rT grub /boot/grub2/themes/weyland-crt

# point GRUB at it (safe to re-run)
sudo sed -i '/^GRUB_THEME=/d' /etc/default/grub
echo 'GRUB_THEME="/boot/grub2/themes/weyland-crt/theme.txt"' | sudo tee -a /etc/default/grub
# MENU FONT — the theme ships three menu-font layouts; use theme.txt above, or swap in one of
# these on that line (then re-run the grub2-mkconfig step) if the entries look too small/off:
#   theme.txt          default  — Share Tech Mono 16
#   theme-bigfont.txt  bigger   — Share Tech Mono 24 (more readable)
#   theme-altfont.txt  alt font — JetBrains Mono 18 (different typeface)

# Nobara ships a text-only menu + hidden menu — turn both off (only needed once)
sudo sed -i 's/^GRUB_TERMINAL_OUTPUT=/#&/' /etc/default/grub
grep -q '^GRUB_GFXMODE=' /etc/default/grub || echo 'GRUB_GFXMODE=auto' | sudo tee -a /etc/default/grub
sudo grub2-editenv - unset menu_auto_hide

# carry GRUB's graphics mode straight into Plymouth so picking a kernel goes cleanly to the
# splash with no black flash (optional; drop this line if Plymouth ever looks garbled).
grep -q '^GRUB_GFXPAYLOAD_LINUX=' /etc/default/grub || echo 'GRUB_GFXPAYLOAD_LINUX=keep' | sudo tee -a /etc/default/grub

# apply (REQUIRED — this also registers the theme font)
sudo grub2-mkconfig -o /boot/grub2/grub.cfg

--------------------------------------------------------------------
2) BOOT SPLASH — install the plugins once, then pick ONE splash
--------------------------------------------------------------------
sudo dnf install plymouth-plugin-script plymouth-plugin-label

# copy in the splash themes (clean install each time)
for t in weyland-muthur weyland-online weyland-nostromo weyland-ignition weyland-console weyland-bar weyland-standby; do
  sudo rm -rf /usr/share/plymouth/themes/$t
  sudo cp -rT plymouth/$t /usr/share/plymouth/themes/$t
done

# choose the one you want (run one of these):
sudo plymouth-set-default-theme -R weyland-muthur      # MU-TH-UR ship computer self-test
# sudo plymouth-set-default-theme -R weyland-online    # emblem powers on + ONLINE
# sudo plymouth-set-default-theme -R weyland-nostromo  # emblem + power gauge energizing
# sudo plymouth-set-default-theme -R weyland-ignition  # minimal: the emblem powers on
# sudo plymouth-set-default-theme -R weyland-console   # plain green boot log
# sudo plymouth-set-default-theme -R weyland-bar       # minimal progress bar
# sudo plymouth-set-default-theme -R weyland-standby   # segmented standby meter
# the -R rebuilds the boot image (~1 min); the splash lives in that image, so
# re-run this line whenever you switch splash or copy new files in.

# make sure the splash is on: check /etc/default/grub has 'quiet' + 'splash'
# (or 'rhgb') between the quotes on a GRUB_CMDLINE_LINUX line; if you edit it,
# re-run the grub2-mkconfig line above.

--------------------------------------------------------------------
3) PLASMA SPLASH — the "Splash Screen" shown AFTER login (KDE)
--------------------------------------------------------------------
# Different from the boot splash above: this is the screen KDE shows while the
# desktop loads, chosen in System Settings > Colors & Themes > Splash Screen.
# No root needed — it installs into your home.
mkdir -p ~/.local/share/plasma/look-and-feel
cp -r plasma-splash/weyland-muthur plasma-splash/weyland-online plasma-splash/weyland-nostromo plasma-splash/weyland-ignition \
      ~/.local/share/plasma/look-and-feel/
kbuildsycoca6 >/dev/null 2>&1   # refresh so they show up in the list

# then either pick it in System Settings > Colors & Themes > Splash Screen
# (hit the little ▶ on a tile to preview first), or set it from the terminal:
kwriteconfig6 --file ksplashrc --group KSplash --key Engine KSplashQML
kwriteconfig6 --file ksplashrc --group KSplash --key Theme weyland-ignition
#   ... or weyland-muthur / weyland-nostromo

# preview any of them right now, no logout needed:
#   ksplashqml --test weyland-ignition

--------------------------------------------------------------------
Reboot to see it.

Test a boot splash without rebooting (from a text console, Ctrl+Alt+F3):
  sudo plymouthd ; sudo plymouth show-splash ; sleep 8 ; sudo plymouth quit
  (Ctrl+Alt+F2 to return to the desktop)

Revert a boot splash:   sudo plymouth-set-default-theme -R bgrt
Revert Plasma splash:   pick "Breeze" (or "None") in the Splash Screen list, and
                        rm -r ~/.local/share/plasma/look-and-feel/weyland-{muthur,nostromo,ignition}
Revert GRUB:      remove the GRUB_THEME and GRUB_GFXPAYLOAD_LINUX lines from
                  /etc/default/grub, un-comment GRUB_TERMINAL_OUTPUT, re-run the
                  grub2-mkconfig line.
